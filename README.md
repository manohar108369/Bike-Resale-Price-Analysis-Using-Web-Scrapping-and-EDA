# Bike Resale Price Analysis Using Web Scraping and EDA

## Overview
This project analyzes 1,274 used bike listings collected from BikeKharido through web scraping.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Project Workflow
1. Web Scraping
2. Data Cleaning
3. Exploratory Data Analysis
4. Insight Generation
5. Dashboard Creation

## Key Insights
- Registration year positively impacts resale price.
- Kilometers driven negatively impacts resale value.
- First-owner bikes dominate the market.
- Metro cities have higher average resale prices.

## Dataset
- 1,274 records
- 8 features
- 54 scraped pages
